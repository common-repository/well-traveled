Plugin to allow access to the Well Traveled tracking script.

== Description ==

This is the plugin that adds the Well Traveled scripts to your page to allow you to track and display audience data.


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `well-traveled.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place your unique user ID in the settings screen.

== Frequently Asked Questions ==

= Where can I find my unique user ID? =

It will be available to you in your setup page https://welltraveled.io/setup to you, if not contact support@welltraveled.io
